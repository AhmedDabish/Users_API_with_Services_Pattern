# UsersApiExample
# simple ASP.NET Core Web API demonstrating the Services Pattern.

Features & Tools Used:

Clean architecture with Controllers, Services, and Repositories.

User management: create, read, and delete users.

Uses Dependency Injection with IUserService interface for flexibility.

Designed for learning, testing, and showcasing best practices in API development.

Tools & Technologies:

.NET 8 / ASP.NET Core Web API

C#

Entity Framework Core (InMemory & SQL Server)

Dependency Injection (DI)

Services Pattern & Repository Pattern
